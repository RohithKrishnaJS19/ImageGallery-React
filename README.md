# Beauty of Kerala - Image Gallery

A responsive and interactive Image Gallery built using React.js and CSS.
This project showcases the scenic beauty of Kerala with smooth UI effects and responsive design.

# Features

• Beautiful image gallery of Kerala tourist places
• Reusable React components
• Hover animation (floating up & down effect)
• Fully responsive design (mobile-friendly)
• Clean and modern UI
• Fast and lightweight

# Tech Stack

• React.js
• CSS3 (Flexbox + Animations)
• JavaScript (ES6)

# How It Works

• Images and data are stored in an array
• A reusable Place component renders each card
• Flexbox is used for layout
• CSS animations provide hover effects
